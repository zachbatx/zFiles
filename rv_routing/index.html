RV Trip Planner

Paste your Google Maps API key to load the map. It is stored only in this browser (localStorage) and never sent anywhere except Google.

How do I get a key?
  1. Go to Google Cloud Console and create a project.
  2. Enable these APIs: Maps JavaScript API, Directions API, Places API, Geocoding API.
  3. Create an API key under Credentials. For personal use you can restrict it to your own referrer later.
  4. Google gives a generous monthly free tier; personal planning rarely exceeds it.
